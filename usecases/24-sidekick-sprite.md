# Sidekick Sprite: The Side Project Guardian

## Persona
Developer with a beloved side project that keeps getting neglected because real life has an alarming number of meetings.

## The Problem They Have
The side repo gathers dust for months, dependencies go stale, and motivation evaporates — until you check the last commit date and it says "4 months ago."

## The NanoClaw Setup
- Scans the repo nightly for outdated dependencies and opens a tidy update PR automatically
- Checks your calendar for free evenings and books 2-hour "code sprint" blocks proactively
- Sends a weekly meme-powered motivation nudge with a progress burndown chart
- Detects inactivity >7 days and pings: "Want to pair-program? Dev container ready."

## Sample Telegram Interaction
> **Agent:** 🐱 Commit streak: 0 days. lodash is 3 versions behind. PR #42 opened. Sprint block booked tonight 8–10pm. Here's a cat in a cape.
> **You:** Let's do this.

## Why It's a Digital Twin
It embodies your side-project zeal, keeping the dream alive even when real work has you hostage.
