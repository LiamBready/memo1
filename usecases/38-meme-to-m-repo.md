# Meme-to-M Repo

## Persona
Content creator who also codes — streams on Twitch, posts dev tutorials, and somehow maintains an actual open-source project.

## The Problem They Have
Great code moments happen live on stream and get lost forever. Blog post ideas come from chat but are forgotten by the time stream ends.

## The NanoClaw Setup
- Listens for "show code" in Twitch/YouTube chat and captures the current snippet automatically
- Opens PRs into stream-demos/ repo so viewers can actually use the code later
- Suggests blog post titles from trending chat topics and saves them to memory
- Sends a midnight digest with top 3 stream moments and a drafted tweet thread

## Sample Telegram Interaction
> **Agent:** 47 "show code" triggers tonight. PR #18 opened with all snippets. Tweet thread drafted: "5 things I learned live on stream tonight." Awaiting your thumbs-up.
> **Dev:** Merge PR, schedule the tweet for 9am.
> **Agent:** Done. Sleep well, creator.

## Why It's a Digital Twin
It captures everything that happens in the moment so nothing good gets lost to the stream archive.
