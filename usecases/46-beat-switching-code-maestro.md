# Beat-Switching Code Maestro

## Persona
Developer who moonlights as a musician -- ships code during the day, plays synth at night, and somehow keeps both identities alive without losing either.

## The Problem They Have
Code sessions bleed into gig prep, gig prep bleeds into code sessions, and the synth sits unplayed for weeks because "I'll do it after this PR."

## The NanoClaw Setup
- Detects free windows after GitHub activity and nudges toward the synth
- Matches musical mood to recent commit messages -- minor key for auth refactors, major after shipping features
- Monitors a wearable for stress spikes to buffer between a tense debugging session and a live gig
- Fires a pre-show checklist 30 minutes before curtain: equipment, setlist, gear charged

## Sample Telegram Interaction
> **Agent:** 6h of commits, mostly error handling. Lo-fi minor 7 feels right. Synth window: 21:00-22:30. Also: gig in 2h -- checklist sent.
> **Dev:** How do you know me this well.
> **Agent:** Break a leg -- and a synth.

## Why It's a Digital Twin
It holds both identities with equal respect so neither one cannibalizes the other.
