# Did You Know? — NanoClaw Facts (Batch 1, #1–5)

---
**Did you know? #1 — CLAUDE.local.md: The Agent's Self-Writing Sticky Note**

In NanoClaw the agent's own playbook lives in a file called CLAUDE.local.md that it can read, edit, and commit just like any other code -- so the agent literally rewrites its own rules while you're still sipping coffee. It's like having a coworker who keeps updating the office manual whenever they spot a typo, except the typo is a missing proactive trigger.

*Source: [What is NanoClaw? — docs.nanoclaw.dev](https://docs.nanoclaw.dev/introduction)*

---
**Did you know? #2 — Proactive AI: Why Waiting for a Prompt Is So 2022**

Reactive agents sit around like bored interns waiting for the boss to say "do something," while proactive NanoClaw agents constantly scan their environment and fire off actions before you even realize you need them. Think of it as the difference between a smoke alarm that beeps when you wave a lighter at it versus one that's already called the fire department.

*Source: [NanoClaw's creators are turning the secure, open source AI agent harness into an enterprise 'second brain' — VentureBeat](https://venturebeat.com/orchestration/nanoclaws-creators-are-turning-the-secure-open-source-ai-agent-harness-into-an-enterprise-second-brain)*

---
**Did you know? #3 — Pre-Check Scripts: The Alarm Clock That Only Rings When Needed**

NanoClaw doesn't waste cycles on a blind timer -- each scheduled task runs a tiny pre-check script first that asks "is there actually work to do?" If the answer is nope, the agent goes back to napping. It's the AI equivalent of setting your phone to buzz only when your mum texts -- everything else can wait.

*Source: [nanocoai/nanoclaw — GitHub](https://github.com/nanocoai/nanoclaw)*

---
**Did you know? #4 — Companion Agents vs One-Shot Agents: The AI That Remembers Your Coffee Order**

A one-shot agent is like a barista who forgets your name after each cup; a companion agent in NanoClaw keeps a persistent memory, so it knows you prefer oat milk, extra shot, and that you hate small talk. This lets the agent build context over weeks and months, turning a fleeting query into an ongoing working relationship.

*Source: [What is NanoClaw? — docs.nanoclaw.dev](https://docs.nanoclaw.dev/introduction)*

---
**Did you know? #5 — Telegram as Agent UI: Why a Chat App Became the Best Control Panel**

Instead of building a bespoke dashboard, NanoClaw hooks straight into Telegram -- so you can issue commands, get alerts, and debug your agent while waiting for the bus. The familiar chat interface means there's exactly zero onboarding, and your proactive digital twin is always one message away, no new tab required.

*Source: [NanoClaw - Secure AI Agent for WhatsApp, Telegram & More — nanoclaw.dev](https://nanoclaw.dev/)*
