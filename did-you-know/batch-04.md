# Did You Know? — NanoClaw Facts (Batch 4, #16–20)

---
**Did you know? #16 — File Delivery Ninja**

NanoClaw agents use send_file to push actual attachments — PDFs, charts, images, generated reports — straight into the chat instead of pasting raw walls of text. It's the difference between a colleague who slides the printed report onto your desk versus one who reads it to you out loud, at full speed, in one message. Your chat stays readable; the artefacts stay accessible.

*Source: [What is NanoClaw? — docs.nanoclaw.dev](https://docs.nanoclaw.dev/introduction)*

---
**Did you know? #17 — The Emoji Whisperer**

With add_reaction, an agent can drop a quick emoji on any message (eyes = seen, check = done) without flooding the thread with a full reply. Think of it as a Slack thumbs-up from someone who is extremely busy but absolutely wanted you to know they saw it — lightweight acknowledgment that respects everyone's attention span.

*Source: [What is NanoClaw? — docs.nanoclaw.dev](https://docs.nanoclaw.dev/introduction)*

---
**Did you know? #18 — Git Blame, But for Your Agent's Brain**

Because the agent's memory lives in plain-text files like CLAUDE.local.md, NanoClaw can commit those files to a GitHub repo after each update — giving you a full version-controlled history of everything the agent has ever learned. Container rebuilt? Memory wiped? Not anymore. You can roll back a bad update or replay a decision from three weeks ago, just like any other piece of code.

*Source: [nanocoai/nanoclaw — GitHub](https://github.com/nanocoai/nanoclaw)*

---
**Did you know? #19 — The Job Description You Can Actually Edit**

NanoClaw encodes agent behaviour in plain Markdown files — CLAUDE.md, CLAUDE.local.md, SKILL.md — not compiled code. Change the file, save it, and the agent immediately starts following the new instructions. It's like handing an employee a revised job description and watching them pick up the new tasks on the spot — no retraining, no recompile, no ticket filed with IT.

*Source: [NanoClaw Guide 2026: Secure, Container-Isolated AI Agents Made Simple — AdvenBoost](https://advenboost.com/nanoclaw/)*

---
**Did you know? #20 — Same Steering Wheel, Different Engine**

Via OpenRouter integration, you can point a NanoClaw agent at any model on the menu — Claude, GPT-4o, Gemini, Llama — without touching a single line of agent logic. Free-tier models let you run surprisingly capable digital twins for near-zero cost, while a premium model is just a config change away. Same steering wheel, same dashboard, same navigation — you just swapped what's under the hood.

*Source: [OpenRouter Docs — openrouter.ai](https://openrouter.ai/docs/overview)*
